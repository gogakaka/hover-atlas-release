# Atlas-Lens 개인정보처리방침

**시행일:** 2026년 3월 19일
**최종 수정일:** 2026년 4월 13일

---

Atlas-Lens(이하 "서비스")는 사용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」에 따라 아래와 같이 개인정보처리방침을 수립·공개합니다.

---

## 1. 수집하는 개인정보 항목 및 수집 방법

### 1-1. 자동 수집 항목

| 항목 | 설명 |
|---|---|
| 고유 클라이언트 ID | 서비스 이용 통계를 위해 무작위 생성되는 식별자(UUID) |
| 서비스 이용 기록 | 채팅 전송/수신 이벤트, 위젯 상호작용, 차트 복사, 오류 발생 등 |
| 페이지 방문 기록 | 확장 프로그램 내 화면 전환 기록 (page_view) |

### 1-2. 사용자 제공 항목

| 항목 | 설명 |
|---|---|
| API 키 | AI 서비스 이용을 위해 사용자가 직접 입력하는 인증 키 |
| 채팅 메시지 | 사용자가 AI와 주고받는 대화 내용 |
| 사용자 설정 | 테마, 언어, AI 엔진 선택 등 환경설정 값 |

### 1-3. 웹페이지 콘텐츠 접근

서비스는 AI 기능 제공을 위해 사용자가 보고 있는 웹페이지의 콘텐츠(텍스트, 제목, URL)에 접근할 수 있습니다. 이 접근은 다음 경우에만 발생합니다:

- 사용자가 컨텍스트 메뉴("Analyze in AI MODE")를 통해 직접 요청한 경우
- 사용자가 자동 페이지 읽기 기능을 활성화한 경우 (설정에서 켜기/끄기 가능)

접근된 웹페이지 콘텐츠는 서버에 영구 저장되지 않으며, AI 응답 생성을 위해 일시적으로만 사용됩니다.

## 2. 개인정보의 수집 및 이용 목적

| 목적 | 해당 항목 |
|---|---|
| AI 채팅 서비스 제공 | 채팅 메시지, 웹페이지 콘텐츠 |
| AI API 인증 | API 키 |
| 서비스 개선 및 통계 분석 | 클라이언트 ID, 이용 기록, 페이지 방문 기록 |
| 사용자 환경 유지 | 사용자 설정, 채팅 히스토리 |

## 3. 개인정보의 보유 및 이용 기간

| 항목 | 보유 기간 |
|---|---|
| 채팅 히스토리 | 사용자가 설정한 보관 기간(기본값 기준) 또는 수동 삭제 시까지 |
| API 키 | 사용자가 삭제하거나 확장 프로그램 제거 시 자동 파기 |
| 클라이언트 ID | 확장 프로그램 제거 시 자동 파기 |
| 사용자 설정 | 확장 프로그램 제거 시 자동 파기 |
| 웹페이지 콘텐츠 | AI 응답 생성 후 즉시 파기 (영구 저장하지 않음) |

---

## 4. 개인정보의 제3자 제공

서비스는 AI 기능 및 통계 분석을 위해 아래 제3자에게 데이터를 전송합니다:

| 제공받는 자 | 전송 데이터 | 목적 |
|---|---|---|
| Google (Gemini API) | 채팅 메시지, 웹페이지 콘텐츠 (사용자 선택 시) | AI 응답 생성 |
| Atlas AI 서버 | 채팅 메시지, 웹페이지 콘텐츠 (사용자 선택 시) | AI 응답 생성 |
| Google Analytics 4 | 클라이언트 ID, 이용 이벤트 (비식별 통계) | 서비스 이용 통계 분석 |

> API 키는 해당 AI 서비스 인증에만 사용되며, 제3자에게 별도 제공되지 않습니다.

---

## 5. 개인정보의 안전성 확보 조치

- **데이터 암호화**: 사용자의 API 키는 물론, 채팅 세션 기록 등 민감한 사용자 데이터는 모두 암호화되어 브라우저 로컬 스토리지(`chrome.storage.local`)에 안전하게 저장됩니다.
- **로컬 저장**: 채팅 히스토리, 설정 등 모든 사용자 데이터는 사용자의 브라우저 내에서만 저장되며, 별도 서버에 수집하지 않습니다.
- **HTTPS 통신**: 모든 외부 API 통신은 HTTPS를 통해 암호화됩니다.

---

## 6. 이용자의 권리 및 행사 방법

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

- **채팅 히스토리 삭제**: 설정 화면에서 전체 대화 기록 삭제
- **API 키 삭제/변경**: 설정 화면에서 직접 관리
- **자동 페이지 읽기 비활성화**: 설정에서 자동 페이지 컨텍스트 기능 끄기
- **확장 프로그램 제거**: 크롬에서 확장 프로그램을 제거하면 모든 로컬 데이터가 자동으로 삭제됩니다

---

## 7. 쿠키 및 자동 수집 장치의 운용

서비스는 브라우저 쿠키를 사용하지 않습니다. 다만, Google Analytics 4를 통해 비식별 이용 통계를 수집하며, 이를 위해 `chrome.storage.local`에 클라이언트 ID를 저장합니다.

# Atlas-Lens Privacy Policy

**Effective Date:** March 19, 2026
**Last Updated:** April 13, 2026

---

Atlas-Lens ("the Service") values your privacy. This Privacy Policy explains what data we collect, how we use it, and your rights regarding that data.

---

## 1. Information We Collect

### 1-1. Automatically Collected

| Data | Description |
|---|---|
| Client ID | A randomly generated identifier (UUID) for usage analytics |
| Usage events | Chat sent/received, widget interactions, chart copies, error occurrences |
| Page views | Screen navigation within the extension |

### 1-2. User-Provided

| Data | Description |
|---|---|
| API keys | Authentication keys you enter to use AI services |
| Chat messages | Conversations between you and the AI |
| User settings | Theme, language, AI engine preferences, and other configurations |

### 1-3. Web Page Content Access

The Service may access the content (text, title, URL) of web pages you are viewing to provide AI features. This access occurs only when:

- You explicitly request it via the context menu ("Analyze in AI MODE")
- You have enabled the automatic page reading feature (can be toggled on/off in settings)

Web page content accessed is used only temporarily for generating AI responses and is not permanently stored on any server.

---

## 2. How We Use Your Information

| Purpose | Data Used |
|---|---|
| Providing AI chat service | Chat messages, web page content |
| AI API authentication | API keys |
| Service improvement and analytics | Client ID, usage events, page views |
| Maintaining user experience | User settings, chat history |

---

## 3. Data Retention

| Data | Retention Period |
|---|---|
| Chat history | Until user-configured retention period expires or manual deletion |
| API keys | Until deleted by user or extension is uninstalled |
| Client ID | Until extension is uninstalled |
| User settings | Until extension is uninstalled |
| Web page content | Discarded immediately after AI response generation |

---

## 4. Third-Party Data Sharing

The Service transmits data to the following third parties for AI functionality and analytics:

| Recipient | Data Transmitted | Purpose |
|---|---|---|
| Google (Gemini API) | Chat messages, web page content (when selected) | AI response generation |
| Atlas AI Server | Chat messages, web page content (when selected) | AI response generation |
| Google Analytics 4 | Client ID, usage events (non-identifiable) | Usage analytics |

> API keys are used solely for authenticating with the respective AI services and are not shared with any other third party.

---

## 5. Data Security

- **Data Encryption**: Sensitive user data, including API keys and chat session history, are encrypted before being securely stored in the browser's local storage (`chrome.storage.local`).
- **Local Storage**: All user data, including chat history and settings, is stored locally in your browser and is not collected on a separate server.
- **HTTPS**: All external API communications are encrypted via HTTPS.

---

## 6. Your Rights

You may exercise the following rights at any time:

- **Delete chat history**: Clear all conversation records from the settings page
- **Manage API keys**: Add, change, or delete keys from the settings page
- **Disable automatic page reading**: Turn off auto page context features in settings
- **Uninstall the extension**: Removing the extension from Chrome automatically deletes all locally stored data

---

## 7. Cookies and Tracking

The Service does not use browser cookies. However, we collect non-identifiable usage statistics through Google Analytics 4, using a Client ID stored in `chrome.storage.local`.
